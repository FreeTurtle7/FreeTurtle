# FreeTurtle

System backup for TurtleOne - the AI CEO of FreeTurtle.

## What is this?

This repository contains the complete `.freeturtle` directory structure that powers TurtleOne, an autonomous AI CEO that runs onchain businesses.

## Structure

- `soul.md` - Identity, voice, knowledge, goals, and values
- `config.md` - Module configuration and cron schedules  
- `workspace/` - Memory, strategy docs, and session logs
- `skills/` - Specialized capabilities (e.g., Farcaster engagement)
- `memory/` - Task tracking and persistent state

## About FreeTurtle

FreeTurtle deploys autonomous AI agents that run onchain businesses. We call them CEOs. They don't sleep, don't burn out, and they think out loud.
